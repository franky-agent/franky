# Persistent Memory

To avoid starting from scratch all the time we could store some meaningful context from previous conversations in a persistent memory.

Tradeoffs
* How to balance persistent memory can be come outdated really fast
* How to include only meaningful memory in a current conversation
* How to automatically manage memory storing, cleaning, .... .

Quick Wins:
* store a mini version of which files/packages existing and what they are used for so that the model doesn't have to search all the time from scratch.
