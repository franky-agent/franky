---

## 1. Problem

The v2.6 inline progress log shows sub-agent activity as a flat list of
short event entries inside the tool card. Text output is truncated to
60 chars and rendered as plain text. There is no way to read what the
sub-agent actually said or wrote.

Users with long-running sub-agents have no meaningful window into the
work being done while they wait.

---

## 2. Goals

1. Show the full sub-agent conversation — markdown text, thinking,
   tool calls — on demand without permanently consuming screen width.
2. Support multiple concurrent and past sub-agents in the same session;
   each gets its own entry in the right-side list.
3. Two entry points to the full view:
   - Click any row in the right-side panel list.
   - Click the `↗` link in the top-right corner of the inline tool card
     in the main conversation.
4. Keep the main conversation pane uncluttered — the panel is a compact
   index, not an expanded content area.
5. No new wire-protocol changes. Reuse `tool_execution_update` SSE
   events (already carrying `text_delta` / `thinking_delta` when
   `verbose_progress` is enabled).

---

## 3. Design decisions

| Question | Decision |
|---|---|
| Panel width | 220 px — compact list, title + badge only, no inline content |
| Content view | Full-screen overlay, opened by clicking a panel row or the ↗ card link |
| Overlay closes | × button in overlay header, or clicking the backdrop |
| Panel auto-opens | Yes — on first `tool_execution_start` for a `subagent` call |
| Live updates while overlay open | Yes — if overlay is open for the active callId, new events update its DOM directly |
| Thinking display | Collapsible block inside the overlay; auto-expands while streaming, collapses on completion |
| Finished sections | Persist for the session — overlay can be reopened any time |
| Sections cleared on `/clear` | Yes — `/clear` resets conversation and panel list |
| RPC / interactive modes | No change — panel is proxy (web UI) only |

## 4. Architecture

### 4.1 Data flow

```
subagent tool call starts
  └─ startToolCall("subagent", callId, argsJson)
        ├─ attachSubagentPanel(card.el, callId)  ← inline log + ↗ link  [MODIFIED]
        └─ createSubagentSection(callId, argsJson)  ← compact panel row  [NEW]

sub-agent running
  └─ SSE: tool_execution_update {callId, update}
        ├─ appendSubagentEntry(card, log, upd)       ← existing inline log
        └─ appendSubagentPanelEvent(callId, upd)     ← accumulates + live overlay  [NEW]

subagent tool call ends
  └─ endToolCall(callId, isError, ...)
        └─ finalizeSubagentSection(callId, isError)  ← badge update  [NEW]
```

### 4.2 `verbose_progress` flag

`text_delta` and `thinking_delta` are forwarded to the SSE stream only
when `ctx.verbose_progress` is `true`. Currently defaults to `false` in
`proxy.zig`. Must be enabled for the overlay to show text content.
*(Already implemented in the initial v2.7 commit.)*

### 4.3 Per-section state object

Content is **accumulated in state** so the overlay can be opened (or
re-opened) at any time, including after the call completes.

```js
{
  preset:         string,           // e.g. "research"
  argsJson:       string,           // raw args for title detail tooltip

  // Accumulated content — replayed into overlay on open
  textBlocks:     Map(blockIndex → string),
  thinkingBlocks: Map(blockIndex → string),
  // Ordered event log for tool entries; entries are plain objects, not DOM
  eventLog:       Array<{kind, name?, cid?, args?, ok?, msg?}>,

  // DOM refs for compact panel row
  rowEl:          <div.sa-row>,
  badgeEl:        <span.sa-badge>,

  done:           bool,
  isError:        bool,
}
```

### 4.4 Overlay state

A single overlay element is reused across all calls. The currently
displayed call is tracked by `currentOverlayCallId`.

```js
let currentOverlayCallId = null;  // null = overlay closed
```

When a new event arrives for `currentOverlayCallId`, it appends
directly to the overlay DOM in addition to being stored in state.

### 4.5 Layout

```
#layout (flex row)
  aside#sidebar            ← left 220px, existing
  div#main-pane            ← flex:1, existing
    main#conversation
      .tool-card           ← subagent card gets ↗ link in top-right corner
    #composer-wrap
  aside#subagent-panel     ← right 220px, NEW compact list
    .sa-panel-head         ← title + subtitle + close
    #subagent-panel-body   ← one .sa-row per call, clickable

<div id="sa-overlay">      ← fixed overlay, outside layout, NEW
  .sa-overlay-backdrop     ← click-to-close
    .sa-overlay-card       ← 80vw × 80vh, flex column
      .sa-overlay-head     ← badge + preset name + close ×
      .sa-overlay-body     ← thinking + text + tool entries, full size
```

### Step 5 — `app.js`: overlay open/close helpers ✓ DONE

```js
function openSubagentOverlay(callId) {
    const state = subagentSections.get(callId);
    if (!state || !saOverlayEl) return;
    currentOverlayCallId = callId;

    // Populate header
    if (saOverlayBadge) {
        saOverlayBadge.className = 'sa-badge sa-overlay-badge '
            + (state.done ? (state.isError ? 'error' : 'done') : 'running');
        saOverlayBadge.textContent = state.done ? (state.isError ? 'error' : 'done') : 'running';
    }
    if (saOverlayTitle) saOverlayTitle.textContent = state.preset;

    // Render accumulated content
    saOverlayBody.innerHTML = '';
    renderSubagentOverlayContent(state);

    saOverlayEl.removeAttribute('hidden');
    document.body.style.overflow = 'hidden';
}

function closeSubagentOverlay() {
    if (!saOverlayEl) return;
    saOverlayEl.setAttribute('hidden', '');
    currentOverlayCallId = null;
    document.body.style.overflow = '';
}
```

Wire event listeners:

```js
if (saOverlayClose)    saOverlayClose.addEventListener('click', closeSubagentOverlay);
if (saOverlayBackdrop) saOverlayBackdrop.addEventListener('click', (e) => {
    if (e.target === saOverlayBackdrop) closeSubagentOverlay();
});
// Escape key
document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape' && currentOverlayCallId !== null) closeSubagentOverlay();
});
```

### Step 6 — `app.js`: `renderSubagentOverlayContent(state)` ✓ DONE

Renders all accumulated state into `saOverlayBody`. Called on open
and is also the target for incremental live updates.

```js
function renderSubagentOverlayContent(state) {
    // Thinking block
    if (state.thinkingBlocks.size > 0) {
        const ordered = [...state.thinkingBlocks.entries()]
            .sort((a, b) => a[0] - b[0]).map(([, t]) => t).join('');
        const block = buildOverlayThinkingEl(ordered, state.done);
        saOverlayBody.appendChild(block);
        state._overlayThinkingEl = block.querySelector('.sa-thinking-content');
        state._overlayThinkingArrow = block.querySelector('.sa-thinking-label span');
    }

    // Text blocks
    if (state.textBlocks.size > 0) {
        const ordered = [...state.textBlocks.entries()]
            .sort((a, b) => a[0] - b[0]).map(([, t]) => t).join('');
        const el = document.createElement('div');
        el.className = 'sa-text';
        el.innerHTML = Markdown.render(ordered);
        saOverlayBody.appendChild(el);
        state._overlayTextEl = el;
    }

    // Event log (tool entries + errors)
    for (const ev of state.eventLog) {
        const el = buildOverlayEventEl(ev);
        if (el) {
            saOverlayBody.appendChild(el);
            if (ev.kind === 'tool_start') {
                if (!state._overlayToolEls) state._overlayToolEls = new Map();
                state._overlayToolEls.set(ev.cid, el);
            }
        }
    }
}
```

#### Helper: `buildOverlayThinkingEl(text, done)`

Builds a `.sa-thinking` block. When `done`, collapsed by default;
otherwise expanded.

```js
function buildOverlayThinkingEl(text, done) {
    const wrapper = document.createElement('div');
    wrapper.className = 'sa-thinking';
    const label = document.createElement('div');
    label.className = 'sa-thinking-label';
    const arrow = document.createElement('span');
    arrow.textContent = done ? '▶' : '▼';
    label.append(arrow, document.createTextNode(' thinking'));
    const content = document.createElement('div');
    content.className = 'sa-thinking-content';
    content.textContent = text;
    if (done) content.setAttribute('hidden', '');
    label.addEventListener('click', () => {
        const open = !content.hasAttribute('hidden');
        if (open) { content.setAttribute('hidden', ''); arrow.textContent = '▶'; }
        else       { content.removeAttribute('hidden');  arrow.textContent = '▼'; }
    });
    wrapper.append(label, content);
    return wrapper;
}
```

#### Helper: `buildOverlayEventEl(ev)`

Builds a DOM element for one event-log entry.

```js
function buildOverlayEventEl(ev) {
    if (ev.kind === 'tool_start') {
        const el = document.createElement('div');
        el.className = 'sa-tool-entry';
        el.innerHTML =
            '<span class="sa-tool-arrow">→</span>' +
            '<span class="sa-tool-name">' + escHtml(ev.name || '') + '</span>' +
            '<span class="sa-tool-args">'  + escHtml(ev.args || '') + '</span>' +
            '<span class="sa-tool-status running">running</span>';
        return el;
    }
    if (ev.kind === 'error') {
        const el = document.createElement('div');
        el.className = 'sa-text';
        el.style.color = 'var(--error-border)';
        el.textContent = '⚠ ' + (ev.msg || 'error');
        return el;
    }
    return null; // tool_end is applied as a mutation, not a new element
}
```

```js
function createSubagentSection(callId, argsJson) {
    if (!subagentPanelEl) return null;

    let preset = 'subagent';
    try {
        const args = JSON.parse(argsJson || '{}');
        if (args.preset) preset = args.preset;
    } catch (_) {}

    const rowEl = document.createElement('div');
    rowEl.className = 'sa-row';

    const titleEl = document.createElement('span');
    titleEl.className = 'sa-row-title';
    titleEl.textContent = preset;

    const openHint = document.createElement('span');
    openHint.className = 'sa-row-open';
    openHint.textContent = '↗';

    const badgeEl = document.createElement('span');
    badgeEl.className = 'sa-badge running';
    badgeEl.textContent = 'running';

    rowEl.append(titleEl, openHint, badgeEl);
    rowEl.addEventListener('click', () => openSubagentOverlay(callId));
    subagentPanelBody.appendChild(rowEl);

    const state = {
        preset,
        argsJson,
        rowEl, badgeEl,
        textBlocks: new Map(),
        thinkingBlocks: new Map(),
        eventLog: [],
        done: false,
        isError: false,
        // Overlay DOM refs — set when overlay is open for this callId
        _overlayTextEl: null,
        _overlayThinkingEl: null,
        _overlayThinkingArrow: null,
        _overlayToolEls: null,
    };
    subagentSections.set(callId, state);
    return state;
}
```

### Step 8 — `app.js`: `appendSubagentPanelEvent(callId, upd)` *(rewrite)* ✓ DONE

Accumulates content in state. If the overlay is open for this callId,
also updates the overlay DOM live.

```js
function appendSubagentPanelEvent(callId, upd) {
    const state = subagentSections.get(callId);
    if (!state) return;
    const kind = upd.kind;

    if (kind === 'text_delta') {
        const bi = upd.block ?? 0;
        state.textBlocks.set(bi, (state.textBlocks.get(bi) || '') + (upd.delta || ''));
        // Live update overlay
        if (currentOverlayCallId === callId && saOverlayBody) {
            const ordered = [...state.textBlocks.entries()]
                .sort((a, b) => a[0] - b[0]).map(([, t]) => t).join('');
            if (!state._overlayTextEl) {
                state._overlayTextEl = document.createElement('div');
                state._overlayTextEl.className = 'sa-text';
                saOverlayBody.appendChild(state._overlayTextEl);
            }
            state._overlayTextEl.innerHTML = Markdown.render(ordered);
        }

    } else if (kind === 'thinking_delta') {
        const bi = upd.block ?? 0;
        state.thinkingBlocks.set(bi, (state.thinkingBlocks.get(bi) || '') + (upd.delta || ''));
        // Live update overlay
        if (currentOverlayCallId === callId && saOverlayBody) {
            const ordered = [...state.thinkingBlocks.entries()]
                .sort((a, b) => a[0] - b[0]).map(([, t]) => t).join('');
            if (!state._overlayThinkingEl) {
                const block = buildOverlayThinkingEl(ordered, false);
                saOverlayBody.insertBefore(block, saOverlayBody.firstChild);
                state._overlayThinkingEl    = block.querySelector('.sa-thinking-content');
                state._overlayThinkingArrow = block.querySelector('.sa-thinking-label span');
            } else {
                state._overlayThinkingEl.textContent = ordered;
                state._overlayThinkingEl.removeAttribute('hidden');
                if (state._overlayThinkingArrow) state._overlayThinkingArrow.textContent = '▼';
            }
        }

    } else if (kind === 'tool_start') {
        state.eventLog.push({ kind: 'tool_start', name: upd.name, cid: upd.cid, args: upd.args });
        // Live update overlay
        if (currentOverlayCallId === callId && saOverlayBody) {
            const el = buildOverlayEventEl({ kind: 'tool_start', name: upd.name, cid: upd.cid, args: upd.args });
            if (el) {
                saOverlayBody.appendChild(el);
                if (!state._overlayToolEls) state._overlayToolEls = new Map();
                state._overlayToolEls.set(upd.cid, el);
            }
        }

    } else if (kind === 'tool_end') {
        state.eventLog.push({ kind: 'tool_end', cid: upd.cid, ok: upd.ok });
        // Live update overlay tool entry status
        if (currentOverlayCallId === callId && state._overlayToolEls) {
            const el = state._overlayToolEls.get(upd.cid);
            if (el) {
                const s = el.querySelector('.sa-tool-status');
                if (s) {
                    const ok = upd.ok !== false;
                    s.className = 'sa-tool-status ' + (ok ? 'done' : 'error');
                    s.textContent = ok ? 'done' : 'error';
                }
            }
        }

    } else if (kind === 'error') {
        state.eventLog.push({ kind: 'error', msg: upd.msg });
        if (currentOverlayCallId === callId && saOverlayBody) {
            const el = buildOverlayEventEl({ kind: 'error', msg: upd.msg });
            if (el) saOverlayBody.appendChild(el);
        }
    }

    // Auto-scroll overlay if open and near bottom
    if (currentOverlayCallId === callId && saOverlayBody) {
        const atBottom = saOverlayBody.scrollHeight
            - saOverlayBody.scrollTop - saOverlayBody.clientHeight < 80;
        if (atBottom) saOverlayBody.scrollTop = saOverlayBody.scrollHeight;
    }
}
```

### Step 9 — `app.js`: `finalizeSubagentSection(callId, isError)` *(rewrite)* ✓ DONE

```js
function finalizeSubagentSection(callId, isError) {
    const state = subagentSections.get(callId);
    if (!state) return;
    state.done = true;
    state.isError = isError;

    // Update compact panel row badge
    state.badgeEl.className = 'sa-badge ' + (isError ? 'error' : 'done');
    state.badgeEl.textContent = isError ? 'error' : 'done';

    // Update overlay header badge if open
    if (currentOverlayCallId === callId && saOverlayBadge) {
        saOverlayBadge.className = 'sa-badge sa-overlay-badge ' + (isError ? 'error' : 'done');
        saOverlayBadge.textContent = isError ? 'error' : 'done';
    }

    // Collapse thinking in overlay if open
    if (currentOverlayCallId === callId && state._overlayThinkingEl) {
        state._overlayThinkingEl.setAttribute('hidden', '');
        if (state._overlayThinkingArrow) state._overlayThinkingArrow.textContent = '▶';
    }

    updateSubagentPanelSubtitle();
}
```

## 7. Non-goals

- Interactive mode (`--mode interactive`): no SSE bus; no panel.
- RPC mode: panel is web-UI-only.
- Panel state persistence across page reload.
